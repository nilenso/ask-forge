# Usage Instructions

1. Install dependencies with `bun install`
2. Import the sandbox runtime with `import { Sandbox } from "@anthropic-ai/sandbox-runtime"`
3. Create and use a sandbox instance to execute code in an isolated environment

# Before Committing

1. Run `bun run check` (auto-fixes most lint and format issues)
2. Manually fix any remaining errors that biome can't auto-fix
3. Commit
